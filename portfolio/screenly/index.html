Screenly

Digital signage made easy.

Worked with a multinational team of engineers to create both a paid and an open source version of Screenly, a distributed digital signage solution that runs on Raspberry Pi hardware.

Screenly Pro consists of a centralized, multi-tenant server cluster which periodically syncs with a thin-client version of Screenly OSE running on clients' Raspberry Pis.
